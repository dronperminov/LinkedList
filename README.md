# LinkedList
Operations for work with templated linked list 

## Operations:
### Getting:
* GetFirst (**O(1)**)
* GetLast (**O(n)**)
* GetAt (**O(n)**)

### Insertion:
* AddFront (**O(1)**)
* AddBack (**O(n)**)
* InsertAt (**O(n)**)

### Removing:
* RemoveFront (**O(1)**)
* RemoveBack (**O(n)**)
* RemoveAt (**O(n)**)
* Remove (**O(n)**)

### Sorting:
* BubleSort (**O(n<sup>2</sup>)**)
* SelectionSort (**O(n<sup>2</sup>)**)
* MergeSort (**O(n log n)**)
